Список изменений
================

+ подтянул VER_C вывод (бывший RAMCS3) на GND (детект версии C платы)
+ использую выводы EPM3064ATC100 для конвертирования напряжения для ША и ШД памяти
+ добавил буфер 74LCX245 для конвертирования напряжения для ША памяти
+ добавил MEMA20 (бывший RAMCS2), MEMA19 
+ сделал память на TC55V8200FT (2x2MB)
+ заменил EPM3032ALC44 на EPM3064ATC100 
+ заменил SD карту на SDC09W4
+ заменил генераторы на OSC7x5

+ на схеме сделал нормальные порты для нетов (с указанием направлений)
+ заменил проц c Z84C0020VEC на Z84C0020FEC
+ L2 заменил на чип дроссель
